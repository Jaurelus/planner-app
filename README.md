# DayFlow

### Description
This is a planner app built to help control the flow of your day by allowing you to plan out events for a day, organized by time. This app also helps keep track of short term and long term goals, allowing you to enter and mark goals off as complete.

### Tech Stack
- Frontend: React Native
- Backend: Node.js, Express.js
- Database: MongoDB

### Problem It Solves
DayFlow bridges the gap between intention and action by transforming abstract goals into a concrete, time-blocked itinerary. This application helps squash procrastination and laziness  by allowing you to keep your daily itenerary, short term goals, and long terms goals all within a few presses of your finger.

### Features
- Allows user to track short term and long term goals with the ability to create, retrieve, update, and delete goals
- Allows user to easily see their daily interary with the ability to create, retrieve, update, and delete tasks from itenerary
- Dynamic timeline to visualize your day as it happens
- Cateogry CLassification: Organize tasks by Physical, Mental, Social, and more to balance your lifestyle
- Calendar Integration - Ability to add marked dates, view holidays
